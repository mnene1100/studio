# **App Name**: NEXO

## Core Features:

- Secure User Authentication: Email/password registration, fast login capabilities, and secure user session management.
- User Onboarding & Numeric ID: A guided onboarding screen for new users to input profile details, automatically generating and displaying a unique 7-9 digit numeric ID on their 'Me' screen.
- Real-time Text Messaging: Core functionality for instant, one-on-one, and group text conversations.
- High-Quality Voice & Video Calls: Enables seamless real-time one-on-one audio and video communication directly within the application.
- AI Conversation Assistant: An AI tool that analyzes chat history to suggest relevant conversation starters or highlight important messages/contacts for a more intelligent interaction experience.
- Intuitive Bottom Navigation: A clear bottom navigation bar allowing users to easily switch between 'Chat' and 'Me' (User Profile) screens.
- Comprehensive User Profile: A dedicated 'Me' screen where users can view, edit, and personalize their profile information and settings.

## Style Guidelines:

- Background color: A deep, almost black, blue-grey (#0D0E0F) to provide a stark, sophisticated canvas as requested.
- Primary interactive color: A rich, deep sapphire blue (#2C628C) that stands out against the dark background while maintaining a premium feel.
- Accent color: A vibrant cyan (#5ED1E9) used for highlights, calls-to-action, and status indicators, offering excellent contrast.
- Text color: Pure white (#FFFFFF) for all primary text, ensuring high readability against the dark background.
- Body and headline font: 'Inter' (sans-serif) for its modern, neutral, and highly legible characteristics, contributing to a clean, premium aesthetic.
- Use sleek, minimalist line icons that align with a modern and premium design philosophy, enhancing clarity and user experience.
- Employ a spacious and clean layout with ample padding around elements to convey a premium and uncluttered user interface, maximizing focus on content.
- Subtle and fluid transition animations for navigation between screens and for call initiation/termination, enhancing the 'premium' user experience with a polished feel.
